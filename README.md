## 🚀 Présentation

Ce portfolio a été développé sans framework ("Vanilla") pour démontrer une maîtrise des standards web actuels. Il met en avant :

* **Architecture Modulaire :** Utilisation du modèle "Bento Grid" (CSS Grid) pour une présentation claire et structurée.
* **Expérience Utilisateur (UX) :** Gestion persistante du thème sombre (Dark Mode) et navigation fluide.
* **Adaptabilité :** Interface entièrement responsive assurant une compatibilité mobile et bureau.

## 🛠 Environnement Technique

* **Front-End :** HTML5, CSS3 (Variables, Flexbox, Grid), JavaScript (ES6).
* **Hébergement :** GitHub Pages.
* **Ressources :** FontAwesome (Icônes), Google Fonts (Outfit & Inter).

## 📂 Contenu du Dépôt

L'arborescence du projet s'articule autour des fichiers suivants :

1.  **`index.html` :** Page d'accueil présentant mon profil et mes compétences transversales.
2.  **`dgfip.html` :** Présentation détaillée de mon alternance à la Direction Générale des Finances Publiques (Missions de support, Scripting Python, Gestion de parc).
3.  **`epreuve-e5.html` :** Documentation technique pour l'épreuve E5 (Projets GSB et missions en entreprise).
4.  **`veille.html` :** Section dédiée à ma veille technologique (Cybersécurité et Green IT). *Note : Section en cours d'enrichissement.*
5.  **`assets/` :** Dossier contenant les ressources graphiques (Images, Logos).
